Supported Exchanges
===================

NexusTrader supports the following exchanges:

- Binance
- OKX
- Bybit
- Bitget
- Hyperliquid
- Bybit TradFi (MT5) — traditional financial markets (Forex, Gold, Indices, Stocks)
